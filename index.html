Introduction

Horus.php is a tiny portable micro php framework that can solve copmlex problems without the need of complex tool kits . Horus.php version 8 is a procedural version, everything is just a function but very organized in just one file, no more words, prepare your editor and start now !

Download Horus

  • From Softpedia
  • From Github Archive
  • From PHPClasses

Setup Horus

  • After downloading horus. unpack it
  • You will notice that it's about three files {Horus.php, index.php, .htaccess}
  • The main file is just Horus.php
  • index.php is just a demo
  • .htaccess is for mod_rewrite Horus can work without .htaccess
  • There are Zero configurations needed to use horus
  • Now open index.php and see the demo

Cheatsheet

Functions

pr()

Alias of print_r() but wrapped into "<pre></pre>"

vd()

Alias of var_dump()

using(mixed $arg)

Useful for chaining
Example: 

    using( new myClass )->get()

errors(bool $state)

enable/disable errors
Example: 

    // hide
    errors(false);
    // show
    errors(true);

settings(mixed $k, [mixed $v])

Application settings wrapper
Example: 

    // set
    settings('key1', 'val1');
    // get
    echo settings('key1');
    // del
    settings('key1', '');
    // all
    vd(settings());
    // add multiple vars
    settings(array('k1' => 'v1', /* ... */));

server(mixed $k, [mixed $v])

Wrapper for $_SERVER array
Example: 

    // set [will convert the key.1 to KEY_1]
    server('key.1', 'val1');
    // get
    echo server('key.1');
    // del
    server('kEy_1', '');
    // all
    vd(server());
    // add multiple env-vars
    server(array('k1' => 'v1', /* ... */));

scope(mixed $k, [mixed $v])

Global vars array
Example: 

    // set
    scope('key1', 'val1');
    // get
    echo scope('key1');
    // del
    scope('key1', '');
    // all
    vd(scope());
    // add multiple vars
    scope(array('k1' => 'v1', /* ... */));

call(callable $callable, [mixed $args])

Alias of call_user_func() and call_user_func_array()
Example: 

    call('func_name', 'arg1');
    call('func_name', 'arg1', 'arg2');
    call('func_name', array('a1', 'a2'));
    call(function($arg){
        echo $arg;
    }, "hi");

tpls(string $filename, [array $vars])

load template(s) file from templates directory
Example: 

    // set the templates dir
    settings('tpls.base', '/path/to/templates_dir');
    // set the templates extension [default is php]
    settings('tpls.ext', 'html');
    // load a template and pass array of vars to it
    tpls('header', array(
        'var1'  => 'val1'
    ));
    // or multiple templates at once !?
    tpls('header,home,footer', $vars);

pdo_conntect(string $dsn, [[string $username], [string $password], [array $driver_options]])

Connect to a database driver using PDO
Example: 

    // connect
    $pdo = pdo_connect("mysql:host=localhost;dbname=test", 'root', '123456');
    // there are some shortcuts for [mysql,postgresql, sqlite]
    $mysql = pdo_connect_mysql($host, $dbname, $username, $password, $options);
    $pgsql = pdo_connect_mysql($host, $dbname, $username, $password, $options);
    $sqlite = pdo_connect_sqlite($dbfile, $options);

pdo_query(PDO $pdo, string $query_string, [mixed $query_inputs])

Run sql statement
Example: 

    // connect
    $pdo = pdo_connect("mysql:host=localhost;dbname=test", 'root', '123456');
    // query
    $query = pdo_query('select * from table');
    // or
    $query = pdo_query('select * from table where col1 = ? and col2 = ?', 'val1', 'val2');
    // or
    $query = pdo_query('select * from table where col1 = ? and col2 = ?', array('val1', 'val2'));

pdo_num_rows(PDOStatement $pdo_statement)

Returns number of rows affetcted by last SQL statement
Example: 

    // connect
    $pdo = pdo_connect("mysql:host=localhost;dbname=test", 'root', '123456');
    // query
    $query = pdo_query('select * from table');
    // the num rows of the $query
    vd(pdo_num_rows($query));

pdo_insert_id(PDO $pdo)

Returns the ID of last inserted row
Example: 

    // connect
    $pdo = pdo_connect("mysql:host=localhost;dbname=test", 'root', '123456');
    // query
    $query = pdo_query('insert into table(col1,col2) values(?, ?)', 'val1', 'val2');
    // the last inserted id of PDO
    vd(pdo_insert_id($pdo));

pdo_fetch(PDOStatement $pdo_statement, [$fetch_style = PDO::FETCH_ASSOC])

Fetch the next row of a result set
Example: 

    // connect
    $pdo = pdo_connect("mysql:host=localhost;dbname=test", 'root', '123456');
    // query
    $query = pdo_query('select * from table');
    // fetch ...
    while($row = pdo_fetch($query)) {
        // do something here
    }

pdo_fetch_all(PDOStatement $pdo_statement, [$fetch_style = PDO::FETCH_ASSOC])

Returns an array containing all of result set rows
Example: 

    // connect
    $pdo = pdo_connect("mysql:host=localhost;dbname=test", 'root', '123456');
    // query
    $query = pdo_query('select * from table');
    // fetch ...
    foreach(pdo_fetch_all($query) as $row) {
        // do something here
    }

hdrs($k = null, $v = null, $replace = true, $code = 200)

header() alternative
Example: 

    // add an header
    hdrs('content-type', 'text/html;charset=utf-8');
    // or [., _ and space] will be '-'
    hdrs('content.type', 'text/html;charset=utf-8');
    // get value of a header key
    vd(hdrs('content-type'));
    // delete
    hdrs('content-type', '');
    // all
    vd(hdrs());

status($new = null)

SET/GET Current http status code
Example: 

    // set to 404
    status(404);
    // get
    vd(status());

out([mixed $args = null])

Append, Override & GET the output
Example: 

    // append
    out('hello', 'world'); // like echo
    // override [set the last argument to true]
    out('hello', 'world', true);
    // get
    vd(out());

respond(string $method, mixed $pattern, callable $callable)

Respond to a browser request
Example: 

    // (1). respond to get request for '/index.html'
    respond('get', '/index.html', function(){
        out("HI, index.html");
    });
    // (2). respond to get or post request for '/page1.html'
    // you can add your own methods [put, delete, ... etc]
    respond('get,post', '/page1.html', function(){
        out("HI, page1.html");
    });
    // (3). respond to any method request for '/page1.html'
    respond('any', '/page1.html', function(){
        out("HI, page1.html");
    });
    // (4). you can use regex
    respond('any', '/([a-zA-Z]+).py', function($name){
        out("HI, ", $name);
    });
    // (5). you can use regex
    respond('any', 'index.py|page.py', function($name){
        out("HI, ", $name);
    });

    // Wait !!, when you goto /index.py or /page.py
    // it will match the 4th and 5th rules !!
    // which one will horus execute ?
    // exactly, the first one only !
    // but if you want to tell horus wait for another one too
    // and don't end now, how to do this ?
    // just return false in the rule !
    // go back to rule-4 and edit it like next:
    respond('any', '/([a-zA-Z]+).py', function($name){
        out("HI, ", $name);
        return false; 
        // will tell horus to also run the next matched one
        // and so for others .
    });

    // but why you did it as that !!
    // > to make Horus more faster, it just execute one route
    //   then end [if needed], so there are no other

    // use some shortcuts in regex ?
    /*
        '@int'      =>  '([0-9\.,]+)',
        '@alpha'    =>  '([a-zA-Z]+)',
        '@alnum'    =>  '([a-zA-Z0-9\.\w]+)',
        '@str'      =>  '([a-zA-Z0-9-_\.\w]+)',
        '@any'      =>  '([^\/]+)',
        '@*'        =>  '?(.*)',
        '@date'     =>  '(([0-9]+)\/([0-9]{2,2}+)\/([0-9]{2,2}+))'
    */

    // you can use any style of supported php callbacks
    // Google for "PHP Callbacks"
    // You can also use a class name or a class object
    // to work like a controller !
    // Example:
    Class Page
    {
        function index()
        {
            // this is the index page '/'
        }
        
        function page1()
        {
            // '/page1/'
        }
        
        function page2($param1, $param2)
        {
            // '/page2/xxx/yyy'
        }
        
        function _page()
        {
            // protected page [404]
        }
    }

    // now respond
    respond('any', '/', 'Page');
    // or
    $page = new Page;
    respond('any', '/', $page);

group($pattern, $callable)

Respond using group of routes (supports unlimited nested groups)
Example: 

    # /api
    group('/api', function(){
        ## api/tweets
        group('/tweets', function(){
            ### api/tweets [insert]
            respond('post', '/', function(){
                // insert code here
            });
            ### api/tweets/ [get all]
            respond('get', '/', function(){
                // fetch all code here
            });
            ### api/tweets/@str [get one by its id]
            respond('get', '/@str', function($id){
                // fetch by id code here
            });
            ### api/tweets/@str [delete a tweet by its id]
            respond('delete', '/@str', function($id){
                // delete code here
            });
            ### api/tweets/@str/re [retweet a tweet by its id]
            respond('post', '/@str/re', function($id){
                // retweet code here
            });
        });
        ## api/users
        group('/users', function(){
            ### api/users [insert]
            respond('post', '/', function(){
                // insert code here
            });
            ### and so ...
        });

    });
?>

bind($event = null, $action = null, $priority = 0)

Bind an action to an event
Example: 

    // bind
    bind('filter.text', $filter_1 = function($text, $last_filtered){
        $text = empty($last_filtered) ? $text : $last_filtered;
        return "{$text}";
    });

    // trigger (run) 'filter.text' events on this word
    // trigger($event, $args = null, $default = null)
    $word = trigger('filter.text', $word = "the word", $word);

    // lets see
    echo $word;

    // dump all events
    vd(bind());

    // remove event(s) [fire]
    fire('filter.text');

    // remove certain action from the events list
    fire('filter.text', $filter_1);

autoload($path, $extension = 'php')

Register given path as spl_autoload_register() implementation
Example: 

    // make the classes with extension 'php' in the directory "/vendors" be autoloaded
    autoload('/vendors', 'php'); // php is the default extension

url(string $format, [mixed $args = null])

Format a local url
Example: 

    /*
        '%route'    =>  server('location.route'),
        '%url'      =>  server('location.url'),
        '%base'     =>  dirname(server('script.name')),
        '%scheme'   =>  server('location.scheme'),
        '%domain'   =>  server('location.domain'),
        '%auto'     =>  server('router.force') ? 'index.php' : "",
        '%path'     =>  ltrim(rtrim(server('router.haystack'), '/'), '/'),
        '%query'    =>  server('query.string'),
        plus PHP format types [%s, %d, ... etc]
    */

    // the direct url to application directory
    vd(url('%url'));
    // the route to application internal route
    vd(url('%route'));
    // custom url formating (1)
    vd(url('%scheme://%domain/%s', 'hi'));
    // custom url formating (2)
    vd(url('%scheme://%domain/%s/%d', 'hi', 5));
    // custom url formating (3)
    vd(url('%scheme://%domain/%s/%d', array('hi', 5)));
    // custom url formating (4)
    vd(url('%scheme://%domain/%auto/%path'));

go($to, $using = 302)

Multi redirect method
Example: 

    // redirect using html [after 0 seconds]
    go('http://google.com', 'html');
    // redirect using html [after 3 seconds]
    go('http://google.com', 'html:3');
    // redirect using javascript [after 0 seconds]
    go('http://google.com', 'js');
    // redirect using javascript [after 5 seconds]
    go('http://google.com', 'js:5');
    // redirect using http 302
    go('http://google.com', 302);
    // redirect using http 301
    go('http://google.com', 301);

    // Note: if you didn't write schema "http://"
    // the function will automatically prepend internal route to it
    // Example:
    go('/api/users');

cls($ob_clean = false)

Clear the browser window
Example: 

    // clean horus output only
    cls();
    // clean horus output and any other output
    cls(true);

start($output_callback = null)

Start horus
Example: 

    // start only
    start();
    // or start with gzip ?
    start('ob_gzhandler');

    // your code here
    
    // end horus
    finish();

Router Class::MethodName filter "hook a filter to the method string"

Example: 

    bind('horus.router.method', function($method_name){
        // when you call /class/method.ext
        // will be /class/method_ext
        return str_replace('.', '_', $method_name);
    });

Override Current Request Method

Example: 

    <form method="post">
        <input hidden="" name="HORUS_OVERRIDE_METHOD" value="put" />
        <input type="submit" />
    </form>
    

    When you submit, it will change the current request method to 'PUT'
    this because browsers don't support except for POST & GET .
